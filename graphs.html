Overview

Reusable pattern library enabling nonvisual access to bar, line, and scatter charts using a “chart description grammar,” keyboard navigation, and optional audio/haptic cues.

Approach

  • Short/long description pairs (title → axes → scale → trend → outliers → takeaway)
  • AT-first interaction model: landmarks, headings, data traversal
  • Iterative tests with blind/low-vision users; refined copy and order

Impact

  • 35–50% faster time-to-insight vs baseline alt-text
  • Higher accuracy on trend/outlier tasks
  • Guidelines and snippets adopted downstream
